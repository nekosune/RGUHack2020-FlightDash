Install FlightDash

FlightDash is playable on Mac, Linux, and Windows, as well in your browser. To download it onto your computer, just follow the instructions below!

  1. Download the installation file.
  2. If prompted, click Run or Save.
  3. If you chose Save, double-click the download to start installing.
  4. Start FlightDash
